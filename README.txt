TraceLens - Intelligent Metadata Analysis and Privacy Inspection Toolkit

Overview
TraceLens is a desktop toolkit for extracting, analyzing, editing, and reporting file metadata.
It provides both a GUI app and a CLI tool.

Included in Installer
- TraceLens.exe (GUI)
- TraceLens-cli.exe (CLI)
- Metadata.png
- License file

Key Features
- Metadata extraction across many file types
- Privacy and forensic risk scoring
- Metadata editing and write-back support
- Report export (PDF, JSON, XML, CSV, Excel)
- Local SQLite history database

System Requirements
- Windows 64-bit

Usage
- Launch GUI: Start Menu > TraceLens
- Launch CLI: Start Menu > TraceLens CLI
